What You Get from Sensei?

With the Sensei plugin you can create courses, write lessons, and add
quizzes. Set lesson and course pre-requisites, allow user registration
and charge.

COURSE PROGESS

A widget for displaying your progress within the current course.

COURSE ANALYTICS

Provides reporting facilities which provide you with an overview of your content, grades.

CHARGE FOR COURSES

Integrates with WooCommerce allowing you to charge for courses.

EASY CONTENT CREATION

Create courses, write lessons, then add quizzes to test your learners

TEST ANYTHING

With a variety of question types at your disposal, create a quizzes.

QUICK USER REGISTRATION

Signing up for a course is super easy with Sensei using WordPress’s user registration.

QUIZ GRADING

Have your simple quizzes graded automatically, or choose manual grading.

QUESTION BANK

Create a ‘bank’ of questions, and display a random selection of them to your learners when they take a quiz.
